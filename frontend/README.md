# Frontend

## Setup

```bash
cd frontend
npm install
```

## Run dev server

```bash
npm run dev
```

Frontend runs at `http://127.0.0.1:5173` by default.

## Run tests

```bash
npm run test
```

## Build

```bash
npm run build
```

## Backend API URL

Create `.env` from `.env.example` if you need a different backend URL.

Default:

```bash
VITE_API_BASE_URL=http://127.0.0.1:8000
```
